INSTRUCTIONS TO USE EXPENSE CHECK FEATURE

STEP 1:

YOU NEED TO CREATE A TABLE AS A LIST-HOLDER TO STORE YOUR MONTHLY EXPENSE DATA.YOU CAN CREATE MULTIPLE TABLE FOR DIFFERENT MONTHS.ITS IMPORTANT TO REMEBRE YOUR TABLE NAME,IT WILL BE REQUIRED FOR LATER TIME IF YOU WOULD WANT TO ADD MORE ITEM TO THAT LIST-HOLDER. NAMING CONVENTION IS AS, USE YOURNAME_MONTH OR ANYTHING MEANINGFUL TO REMEMBER.NOTE: DONT USE SPACE OR DASH BETWEEN 2 WORDS AS TABLE NAME.

STEP 2:

ONCE YOU CRETAED YOUR DATA TABLE,IT CREATES A EMPTY LIST HOLDER IN THE DATABASE,INITIALLY THE TOTAL AMOUNT WILL BE SHOWN AS ZERO.

STEP 3:

NOW YOU CAN ADD ANY EXPENSE ITEM WITH RESPECTED DATE IN YOUR TABLE.NOTE:YOU MUST PROVIDE THE EXACT TABLE NAME TO INSERT NEW ITEMS AND GET THE RESULT. YOU MAY USE MY TABLES FEATURE TO VIEW YOUR TABLE SO YOU WONT NEED TO TYPE THE TABLE NAME IN TABLE FIND FORM MANUALLY.

STEP 4:

ONCE YOU ARE DONE,YOU CAN COME BACK LATER AND FETCH THE DATA FROM YOUR TABLE USING FIND TABLE BUTTON PROVIDING EXACT TABLE NAME AND HIT FIND TABLE BUTTON.OTHERWISE, EASIER WAY IS TO SEE ALL YOUR TABLES UNDER MY TABLES SECTION AND CLICK CHECKBOX TO ANY SPECIFC TABLE AND YOU MAY VIEW YOUR DATA.

STEP 5:

YOU HAVE ABILITY TO MODIFY OR UPDATE EITHER YOUR TABLE UNDER MY TABLES SECTION OR ANY RECORD WITH IN ANY TAVLE USING EDIT,DELETE OPTION.NOTE: SELECT ONE SPECIFIC CHECKBOX TO PERFORM UPDATE OPERATION.

STEP 6:

YOU ARE ALL SET.YOU ARE ABLE TO USE THIS APP AS MUCH AS YOU WANT.N ORDER TO STORE AND KEEP TRACK OF YOUR MONTHLY EXPENSE,YOU CAN NOW HAVE ANY NUMBER OF TABLE FOR DIFFERENT MONTHS OR EVENTS AND YOU CAN ADD AS MANY ITEMS AS YOU WANT.
