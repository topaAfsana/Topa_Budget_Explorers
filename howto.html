INSTRUCTIONS TO USE EXPENSE CHECK FEATURE

STEP 1:

YOU NEED TO CREATE A TABLE AS LIST-HOLDER TO STORE YOUR MONTHLY EXPENSE DATA.YOU CAN CREATE MULTIPLE TABLE FOR DIFFERENT MONTHS.ITS IMPORTANT TO REMEBRE YOUR TABLE NAME,IT WILL BE REQUIRED LATER TIME YOU WOULD WANT TO ADD MORE ITEM TO THAT TABLE/LIST. NAMING CONVENTION IS AS, USE YOURNAME_MONTH OR ANYTHING MEANINGFUL TO REMEMBER AND FIND OF YOURS ONE FOR LATER TIME.

STEP 2:

ONCE YOU CRETAED YOUR DATA BASE TABLE,IT CREATES A EMPTY LIST HOLDER IN THE DATABASE,THE TOTAL AMOUNT WILL BE SHOWN AS ZERO INITIALLY.

STEP 3:

NOW YOU CAN ADD ANY EXPENSE ITEM WITH RESPECTED DATE IN THE TABLE YOU CREATED.NOTE:YOU MUST PROVIDE THE EXACT TABLE NAME TO INSERT NEW ITEMS AND GET THE RESULT

STEP 4:

ONCE YOU ARE DONE, CAN COME BACK LATER AND FETCH THE DATA FROM YOUR TABLE USING FIND TABLE.ALL YOU HAVE TO DO IS ENTER THE EXACT NAME OF YOUR TABLE AND EITHER HIT FIND TABLE TO VIEW THE CURRENT TABLE OR MODIFY BY ADDING NEW EXPENSE ITEMS USING ADD ITEM OPTION.

STEP 5:

YOU CAN USE THIS FEATURE AS MUCH AS YOU WANT ,AS MANY ITEM YOU WANT ,IN ORDER TO STORE AND KEEP TRACK OF YOUR MONTHLY EXPENSE.
